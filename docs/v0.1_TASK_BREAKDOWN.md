## 子任务分解

### T1: Flutter 项目初始化

**交付内容**：Flutter 项目 Android + iOS 双平台配置，目录结构搭建

| 步骤 | 操作 | 产出 |
|------|------|------|
| 1.1 | 在 `/home/hhikr/ArkLores` 内执行 `flutter create` | `pubspec.yaml`, `android/`, `ios/`, `lib/main.dart` |
| 1.2 | 按 `implementation_plan.md` 目录结构调整 `lib/` 结构 | `lib/core/`, `lib/features/`, `lib/shared/` 骨架目录 |
| 1.3 | 配置 `pubspec.yaml`，添加首批依赖 | `flutter_riverpod`, `google_fonts` 等 v0.1 所需包 |
| 1.4 | 运行 `flutter pub get` 并验证 | 无报错 |

**commit**: `chore(project): initialize Flutter project with directory structure`

---

### T2: 双主题 Token 系统

**交付内容**：`ArkThemeTokens` + `EndfieldThemeTokens`，设置页可切换

| 步骤 | 操作 | 产出 |
|------|------|------|
| 2.1 | 创建抽象 `AppThemeTokens` | `lib/shared/theme/app_theme.dart` |
| 2.2 | 实现 `ArkThemeTokens`（战术档案风配色） | `lib/shared/theme/ark_theme_tokens.dart` |
| 2.3 | 实现 `EndfieldThemeTokens`（全息投影风配色） | `lib/shared/theme/endfield_theme_tokens.dart` |
| 2.4 | 创建 `ThemeNotifier` + `themeProvider`（Riverpod StateNotifier） | `lib/shared/providers/theme_provider.dart` |
| 2.5 | 在 `lib/app.dart` 中接入 `ThemeNotifier`，使用 `ProviderScope` | App 根节点注入 theme |
| 2.6 | 验证：切换 Provider 值时主题色变化 | 手动测试 |

**commit**: `feat(theme): implement dual theme token system with ArkTheme and EndfieldTheme`

---

### T3: 底部导航四 Tab 框架

**交付内容**：Wiki / AI / 资料 / 设置 四个 Tab，空页面占位

| 步骤 | 操作 | 产出 |

| 3.1 | 创建 4 个占位页面 | `WikiBrowserPage`, `AiChatPage`, `MaterialsPage`, `SettingsPage` |

| 3.4 | 验证：4 Tab 切换正常，页面内容随主题变化 | 手动测试 |

**commit**: `feat(theme): add bottom navigation with four tabs`

### T4: 字体与基础组件

**交付内容**：Rajdhani / Exo 2 / Noto Sans SC 载入，`ThemeAwareCard` 组件

| 步骤 | 操作 | 产出 |
|------|------|------|
| 4.1 | 在 pubspec.yaml 中配置 `google_fonts` | 字体包加载 |
| 4.2 | 在 `AppThemeTokens` 中定义 `titleFont` / `bodyFont` getter | 每种主题指定对应字体 |
| 4.3 | 创建 `ThemeAwareCard` 组件 | `lib/shared/widgets/theme_aware_card.dart` |
| 4.4 | 在每个占位页面中展示 `ThemeAwareCard` + 主题字体 | 验证 UI |

**commit**: `feat(theme): add Google Fonts integration and ThemeAwareCard component`

**交付内容**：设置页内可切换主题，切换时全屏 fade 过渡（300ms）

| 步骤 | 操作 | 产出 |

**commit**: `feat(settings): add theme switcher with fade transition animation`

│       └── settings_page.dart          # 主题切换开关

        └── theme_provider.dart         # ThemeNotifier + provider
