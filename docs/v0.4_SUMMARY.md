4. **AI 对话界面骨架与交互集成**：
   - 重构了 [ai_chat_page.dart](file:///home/hhikr/ArkLores/lib/features/ai/ai_chat_page.dart)，添加了顶部 TabBar 切换三个模式，并使用 `DefaultTabController` 自定义冷钢蓝/青色主题样式。
   - 编写了 [chat_bubble.dart](file:///home/hhikr/ArkLores/lib/features/ai/widgets/chat_bubble.dart) 和 [lazy_citation_card.dart](file:///home/hhikr/ArkLores/lib/features/ai/widgets/lazy_citation_card.dart)，支持：
     - 在助手气泡上方显示折叠的 ReAct 决策过程（如：Thought 过程，使用 search_wiki/cite_source 工具的参数及 observation 回馈），使用户能透明直观地看到 AI 的检索历程。
     - **智能脚注匹配**：对 AI 回复中生成的 `[UUID]` 引用标记，使用 RegExp 动态扫描并转换为轻量级的 Markdown Footnote 标号（如 `[^1]`），同时在气泡下方惰性渲染 `LazyCitationCard`（在 sqlite 数据库中检索对应 ID 并生成带 Wiki/Book 颜色区分和动效的 `CitationCard`）。
     - 支持无 API 密钥时的友好提示与快捷前往设置页配置，以及快捷清空历史对话确认框。

5. **国际化与单元测试**：
