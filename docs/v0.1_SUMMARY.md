## 创建的文件清单

### 核心入口
- `lib/main.dart` — 应用入口，ProviderScope 包裹，ArkLoresApp 组件
- `lib/app.dart` — MainShell：底部导航 + IndexedStack + AnimatedSwitcher 300ms fade

### 主题系统 (`lib/shared/theme/`)
- `app_theme.dart` — `AppThemeTokens` 抽象接口（颜色、字体、几何、语义色共 17 个 Token）
- `ark_theme_tokens.dart` — 主题A：战术档案风（冷蓝灰调，#0B0D10 背景，#4A90D9 强调）
- `endfield_theme_tokens.dart` — 主题B：全息投影风（宇宙蓝调，#050810 背景，#00C8FF 强调）

### 状态管理 (`lib/shared/providers/`)
- `theme_provider.dart` — `ThemeNotifier` + `themeProvider`（Riverpod StateNotifier），支持 `switchTo()` / `toggle()` / `currentTheme`

### 基础组件 (`lib/shared/widgets/`)
- `theme_aware_card.dart` — 自适应主题卡片，读取 themeProvider 获取所有视觉 Token

### 四个 Tab 占位页面
- `lib/features/wiki/wiki_browser_page.dart` — Wiki 浏览（v0.2 实现）
- `lib/features/ai/ai_chat_page.dart` — AI 对话（v0.4 实现）
- `lib/features/materials/materials_page.dart` — 资料管理（v0.3 实现）
- `lib/features/settings/settings_page.dart` — 设置（含主题切换开关，完整功能 v0.3）

### 其他
- `test/widget_test.dart` — 更新为 ArkLoresApp 的 smoke test（检测 4 Tab 渲染）

## 需要人工验证的检查点

1. **在 Android 模拟器上运行** — 确认 4 Tab 可正常切换，无渲染错误
2. **主题切换视觉效果** — 切换到设置页，拨动 Switch 开关，确认：
   - 颜色从冷蓝灰（Ark）变为全息青（Endfield）
   - 字体从 Rajdhani 变为 Exo 2
   - 切换时有 300ms fade 过渡动画
3. **ThemeAwareCard 样式** — 确认每个 Tab 页中的卡片样式跟随主题变化
4. **Google Fonts 首次加载** — 首次运行需网络下载字体，确认字体加载后正确渲染
5. **Android 最低版本兼容性** — 确认 API 26 无兼容性问题
